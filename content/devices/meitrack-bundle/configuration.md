To connect a Meitrack device to Cumulocity, you need to configure it to send data to the Cumulocity servers. Use the Meitrack manager to specify the following settings:

* **GPRS protocol**: TCP
* **IP/Domain**: tracker.cumulocity.com
* **Port** : 9091
