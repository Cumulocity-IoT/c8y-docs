In this section, [Hello MQTT](#hello-mqtt) provides an easy introduction to the {{< product-c8y-iot >}} MQTT protocol using a MQTT cross platform web app.

It also contains examples on how to use the MQTT client with {{< product-c8y-iot >}} employing pre-defined messages (called "static templates") using [C](#hello-mqtt-c), [Java](#hello-mqtt-java), [JavaScript](#hello-mqtt-javascript), [Python](#hello-mqtt-python) and [C#](#hello-mqtt-cs).

All the examples can be downloaded from the [GitHub repository](https://github.com/SoftwareAG/c8y_hw_mqtt).
