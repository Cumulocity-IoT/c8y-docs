Cumulocity IoT Edge Release 10.5, November 2019, includes the following enhancements and changes:

### Apama

* Cumulocity IoT Edge Release 10.5 includes Apama 10.5.0.
* If you are upgrading from Cumulocity IoT Edge version 10.4 to 10.5, a new developer experience is available when editing and deploying Apama EPL apps. See, [Apama streaming analytics - New code editor and manager](/guides/release-notes/10-4-6/#apama-streaming-analytics-new-code-editor-and-manager).
