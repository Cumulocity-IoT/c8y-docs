### Apama correlator version

This release of Cumulocity IoT Streaming Analytics includes the Apama version 10.7.1 correlator.

### New Streaming Analytics application

A new application called Streaming Analytics is now available in the Cumulocity IoT application switcher. The Apama Analytics Builder and Apama EPL Apps applications from previous versions are now part of this new Streaming Analytics application. See the description of the new applications's [home screen](/apama/overview-analytics/#home) for more information.
