Improvements

Area Description
Enterprise Edition SSL management error text improvements.
Enterprise Edition The Tenant Statistics page now presents information on microservice usage per tenant.
Enterprise Edition The support user feature has been extended. It is now possible to deactivate the support user and to see support access status in the subtenants details.
Device Management Error messages from shell commands are now more user-friendly.
LWM2M It is now possible to write binary resources to devices.
MQTT Support for pre-defined topics has been added.
MQTT It is now possible to connect registered devices via MQTT-SN gateway.
LoRa/UI Improvements for validation error texts.
