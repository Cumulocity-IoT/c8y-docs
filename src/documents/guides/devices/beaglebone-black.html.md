## Overview

[BeagleBone - Black](https://beagleboard.org/black) is a low-cost, community-supported development platform perfect for IoT applications. The device features the following interfaces:
> 1 x 10/100 MBit/s Ethernet

> 1 x USB 2.0
 
> 1 x HDMI
