SmartREST 2.0 lets you create template for the following matching HTTP Methods:

|API|GET|POST|PUT|
|:-------|:-------|:-------|:-------|
|Inventory|x|x|x|
|Alarm||x|x|
|Event||x||
|Measurement||x||
|Operation|||x|

Additionally you can create templates to return certain values from responses and operations.
