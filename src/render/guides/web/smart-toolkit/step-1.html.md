Create the following folder structure for the project:

```
hello-core-api
├── css
├── js
│   ├── alarms_ctrl.js
│   ├── app.js
│   ├── login_ctrl.js
│   ├── main_ctrl.js
│   └── section_dir.js
├── sections
│   ├── alarms.html
│   ├── devices.html
│   └── events.html
├── bower.json
├── Gruntfile.js
├── index.html
├── login.html
├── main.html
└── package.json
```

Copy the CSS files from [the example project](https://bitbucket.org/m2m/cumulocity-examples) under the folder "hello-core-api/css".
